# weather-counsel
a convenient tool for searching weather forecast
## setup
```
$ npm install weather-counsel -g
```
## command
show the weather forecast of your city
```
$ weather
```
show the weather forecast your assign
```
$ weather city
```
result example
```
$ weather
天气：多云
气温：31 ~ 24℃
风向：南风微风
pm2.5：103
```
